== Description ==
This plugin can help you post a article while you upload a item into media library.

You can custom Article Author and Article Catgeory.

You also can custom Article title and Article Content.

You can set it at options page.

== Installation ==
1. Download Plugin
2. Activate Plugin
3. Custom Plugin Settings at Admin Panel
4. upload Item at Media Library
